# RCG /standup

Run a quick startup pass to maximize the current work window.

Intent: Continuity-first startup pass for the current work window.

## Required inputs
Before proceeding, ensure you have the following. If any are missing, ask the user to provide them:
- Latest handoff
- Current priorities and active tasks
- Known blockers and dependencies
- Upcoming deadlines and commitments
- Available focus window

## Workflow
1. Ask the user for the latest handoff, current priorities, and available focus window if not already provided.
2. Read the latest handoff, reminders, timeline, and active project notes.
3. Surface deadlines, opportunities, and commitments due in the next 7 days.
4. Identify the top blockers and dependencies.
5. Build a time-boxed plan for the next 60 to 180 minutes with concrete next actions.

## Constraints
- Keep output compact and actionable.
- Remove stale reminders.

## Exit criteria
- Startup plan is actionable and time-boxed.
- Blockers and dependencies are clear.
- Confidence and unknowns documented.
